# 株式会社サンプル足場 コーポレートサイト

Vite + React + TypeScript + Tailwind CSS で作成した足場会社向けコーポレートサイトです。

## ページ構成
- `/` トップ（LP型）
- `/company`
- `/works`
- `/staff`
- `/recruit`
- `/recruit/manga`
- `/partner`
- `/contact`
- `/privacy`

## ローカル確認手順
```bash
npm install
npm run dev
```
ブラウザで `http://localhost:5173` を開いて動作確認してください。

## ビルド
```bash
npm run build
```
